# JPG 轉 PDF（純前端、免上傳）

一個 **純前端、可離線使用** 的 JPG / PNG 轉 PDF 工具。  
支援多張圖片拖曳排序、旋轉、調整輸出品質，所有處理都在瀏覽器中完成，圖片不會上傳到任何伺服器。

👉 線上使用：  
https://suwaillya.github.io/jpg-to-pdf/

---

## ✨ 功能特色

- 📂 **多張圖片轉 PDF**
  - 支援 JPG / PNG
  - 可在選擇後繼續加入新圖片

- 🖱️ **拖曳排序（桌機 & 手機）**
  - 類似 iLovePDF 的拖曳體驗
  - 拖曳時圖片會「被拉起來」
  - 支援自動捲動，圖片多也好操作

- 🔄 **圖片旋轉**
  - 每張圖片可獨立旋轉（90°）
  - 旋轉效果會正確反映在 PDF 輸出中

- 🗑️ **即時刪除圖片**
  - 單張移除，不影響其他順序

- 📄 **PDF 輸出設定**
  - 頁面大小：A4 / Letter / Fit（依圖片裁切）
  - 方向：直向 / 橫向 / 自動
  - 邊距（mm）
  - 壓縮品質（0.1 ~ 1）

- 🔐 **隱私友善**
  - 純前端處理
  - 不上傳、不儲存、不追蹤

---

## 🧠 為什麼做這個工具？

市面上許多圖片轉 PDF 服務需要：
- 上傳檔案
- 登入帳號
- 或有檔案大小限制

這個專案的目標是提供一個：
- **簡單**
- **透明**
- **可自行部署**
- **不依賴後端**

的圖片轉 PDF 解決方案。
